<h2>##2. Description of the project.</h2>
=> The Website is made for a pet adoption platform named by PEDDY. Each adoptable pet will have a detailed profile, including their breed, age, size, temperament, health status, and photos, so that potential adopters can make informed decisions.

<h2>##3. 5 key features of the project.</h2>
   <ul>
        <li>All category Selection</li>
        <li> Details Features Added</li>
        <li>Like Features Added</li>
        <li> Spinner Features Added</li>
        <li> Sorting Price</li>
  </ul>

<h2>##4. ES6 features used.</h2>   
   <ul>
       <li> forEach(): PEDDY  loops through pets and categories, displaying the results in the browser.</li>
       <li>async/await: Handles asynchronous operations such as fetching pet data from the API></li>
       <li>sort(): Sorts pets in descending order according to user preferences, such as price</li>
   </ul>
